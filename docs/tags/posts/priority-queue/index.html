Priority queue

Priority queue is similar to normal queue or stack, except every element has a priority. Priority queue is usually implemented as heap.

Heap

Heap is a tree data structure, it is almost complete binary tree. Common implementation of heap is the binary heap.

left child 2i+1

right child 2i+2

parent i / 2 - 1
