Bitwise logical operators

Bitwise AND (&)

Compares every bit of two numbers and applies AND operation to each of them. For example 5 (101b) and 3 (011b) will give 1.

Even or odd

Even numbers end with 0 on last bit position, odd has 1 there. For example

0001 - 1
0010 - 2
0011 - 3
0100 - 4
0101 - 5

Simple code to check if number is odd or even

return (n & 1) == 1; //true if odd
